## 82
- logitsはsoftmax関数で確率化
- dimはPyTouchテンソル演算でどの次元に対しての処理なのかを指すパラメータ(語彙次元が[1,約3万]で配列index0初めから見ると約3万に対する処理)
- 尤度(Likelihood)は，機械学習でモデルが観測データにどれだけ適合するか
### softmax関数
$$
\text{softmax}(Z)_{i,j} = \frac{e^{Z_{i,j}}}{\sum_{k=1}^{V} e^{Z_{i,k}}}
$$
